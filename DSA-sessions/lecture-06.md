# Lecture 6: The Recommendation Problem

- What is recommendation? Systems that infer interest from data
- Data types: ratings, text (reviews), images (posters) — each with its own representation challenges
- Rating matrices: explicit vs. implicit feedback
- Core problems: sparsity, cold start, long tail / popularity bias
- Evaluation dimensions: relevance, diversity, serendipity, explainability
